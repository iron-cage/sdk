# Admin Manual
Manual on how to use the iron cage dashboard as Admin.

### Dashboard
On this page, the admin can see all metrics about the iron cage tokens. For example, `Total Tokens`, `Active Tokens`, `Revoked Tokens`, `Recent Tokens`.

<img width="1917" height="902" alt="image" src="https://github.com/user-attachments/assets/a161f312-056c-4d33-8a60-7e8f6582a5f6" />

### Agents
On this page, the admin can manage agents, generate IC tokens, and link LLM to any agent from the list.

<img width="1919" height="903" alt="image" src="https://github.com/user-attachments/assets/f7d6fd43-219a-464d-be8f-851f05d3abad" />

### Analytics
On this page, the admin can see all information about the agent's use. For example, `Total Requests`, `Success Rate`, `Input Tokens`, `Output Tokens`, `Total Cost`, `Agent Provider`, `Usage by Model`.

<img width="1917" height="901" alt="image" src="https://github.com/user-attachments/assets/a2398db9-f37f-402e-a45c-39d2fcdac9e0" />

### Budget
On this page, the  admin can manage agent limits by money.

<img width="1919" height="907" alt="image" src="https://github.com/user-attachments/assets/6cf1db54-a64a-4c61-b7f3-0828ae0bcb17" />

### Providers
On this page, the admin can manage all AI providers' keys. For example, add a new one or disable the existing key.

<img width="1919" height="902" alt="image" src="https://github.com/user-attachments/assets/e2e400f1-b49f-4865-bf36-46ccb519fc77" />

On this page, the admin can manage all users of the project, filter them, update roles, delete users, create users, and reset user passwords.
