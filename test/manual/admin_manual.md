# Admin Manual
Admin manual for the iron cage using

### Dashboard
On this page, the admin can see all metrics about the iron cage tokens. For example, `Total Tokens`, `Active Tokens`, `Revoked Tokens`, `Recent Tokens`.

### Agents
On this page, the admin can manage agents, generate IC tokens, and link LLM to any agent from the list.

### Usage Analytics
On this page, the admin can see all information about the agent's use. For example, `Total Requests`, `Success Rate`, `Input Tokens`, `Output Tokens`, `Total Cost`, `Agent Provider`, `Usage by Model`.

### Limits
On this page, the  admin can manage project/agent limits by tokens, money, or requests.

### Budget Requests
On this page, the admin can see, approve, and create a budget request. This page is used to create a request for the agent budget update.

### AI Providers
On this page, the admin can manage all AI providers' keys. For example, add a new one or disable the existing key.

### Users
On this page, the admin can manage all users of the project, filter them, update roles, delete users, create users, and reset user passwords.
