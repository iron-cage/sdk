# Iron Agents

Container module for agent implementations and examples.

[![Documentation](https://img.shields.io/badge/docs-ironcage.ai-E5E7EB.svg)](https://ironcage.ai/docs)

## Structure

```text
.
└──  lead_generator_agent          # Lead generator agent module
```


## Purpose

This module contains:
- **Reference Implementations**: Production-ready agents demonstrating best practices
- **Integration Examples**: Agents using various frameworks (LangChain, CrewAI, AutoGPT)
- **Starting Templates**: Copy-paste examples for new agent development


## Contributing New Agents

When adding new agent modules:

1. Create subdirectory with descriptive name (e.g., `email_analyzer_agent`)
2. Include readme.md with agent purpose and configuration
3. Add example to this directory listing
4. Document any framework-specific integration patterns
